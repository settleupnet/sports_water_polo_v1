# Sports Water Polo V1

Water Polo sport engine v1 for QuickCap.

Provides validation, derivation, and presentation logic for Water Polo events.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sports_water_polo_v1', git: 'https://github.com/settleupnet/sports_water_polo_v1.git'
```

Or for local development:

```ruby
gem 'sports_water_polo_v1', path: 'vendor/sports/water_polo/v1/ruby'
```

And then execute:

```bash
bundle install
```

## Usage

This gem provides sport-specific logic for Water Polo events in the QuickCap platform.

## Development

After checking out the repo, run:

```bash
bundle install
rspec
```
